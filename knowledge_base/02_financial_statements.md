# 财务报表解读 (Financial Statement Analysis)

## 收入与净利润的区别

### 收入 (Revenue / 营业收入)

收入是公司在一定期间内通过核心业务活动获得的总金额，也称为"营业收入"或"销售额"。

**特点：**
- 位于利润表最顶部，因此也被称为 "Top Line"（顶线）。
- 反映公司的市场规模和业务增长能力。
- 不扣除任何成本和费用。

### 净利润 (Net Income / 净收入)

净利润是公司在扣除所有成本、费用、税金和其他支出后的最终利润。

**特点：**
- 位于利润表最底部，因此也被称为 "Bottom Line"（底线）。
- 反映公司最终的盈利能力。
- 包含一次性收益和费用的影响。

### 从收入到净利润的路径

```
营业收入 (Revenue)
  - 营业成本 (COGS)
= 毛利润 (Gross Profit)
  - 运营费用 (Operating Expenses)
    - 销售及管理费用 (SG&A)
    - 研发费用 (R&D)
    - 折旧和摊销 (D&A)
= 营业利润 (Operating Income / EBIT)
  + 其他收入 / - 其他支出
  - 利息费用
= 税前利润 (Pre-tax Income)
  - 所得税
= 净利润 (Net Income)
```

### 关键差异

| 维度 | 收入 | 净利润 |
|------|------|--------|
| 位置 | 利润表顶部 | 利润表底部 |
| 含义 | 总业务收入 | 最终盈利 |
| 扣除项 | 无 | 所有成本和税费 |
| 增长快？ | 不一定 | 取决于成本控制 |

## 毛利率 (Gross Margin)

**计算公式：**
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%

**解读：**
- 反映公司核心产品或服务的盈利能力。
- 高毛利率通常意味着较强的定价能力或较低的生产成本。
- 软件和互联网公司通常毛利率较高（60%-80%），制造业较低（20%-40%）。

---

## 营业利润率 (Operating Margin)

**计算公式：**
营业利润率 = 营业利润 / 营业收入 × 100%

**解读：**
- 衡量公司在扣除营业费用后的经营效率。
- 排除了利息和税收的影响，更能反映核心业务盈利能力。
- 持续下降的营业利润率可能暗示成本控制问题。

---

## 净利率 (Net Margin)

**计算公式：**
净利率 = 净利润 / 营业收入 × 100%

**解读：**
- 反映公司从每一美元收入中最终保留的利润比例。
- 高净利率意味着公司具有较强的整体盈利能力。
- 行业差异巨大：科技公司可达 20-30%，零售业可能仅 2-5%。

---

## 自由现金流 (Free Cash Flow)

**计算公式：**
自由现金流 = 经营活动现金流 - 资本支出 (CapEx)

**重要性：**
- 自由现金流代表公司在维持和扩展业务后可自由支配的现金。
- 正的自由现金流意味着公司可以回购股票、发放股息或偿还债务。
- 自由现金流可能与净利润显著不同（因折旧、应收账款等非现金项目）。
- Warren Buffett 经常强调自由现金流比净利润更能反映公司真实价值。
